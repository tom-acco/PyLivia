# PyLivia
A Python chat application implementing the [Olivia](https://en.wikipedia.org/wiki/Olivia_MFSK) protocol.

# Setup
## Install
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Device Listing
```bash
python -m sounddevice
```

## Configure
